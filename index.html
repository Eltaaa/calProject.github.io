คำอธิบายโครงงาน โปรเจกต์นี้เราใช้ Keypad 4x4 เป็นอุปกรณ์หลักในการรับข้อมูลตัวเลข ใช้ปุ่ม A B C D แทน + - * / แล้วใช้ปุ่ม # ในการทำการคำนวณ กดปุ่ม * เพื่อล้างการป้อนค่า ด้วยความที่เราใช้ไปแล้ว 16 ปุ่ม ทำให้เครื่องคิดเลขที่เราทำขึ้นมา ไม่สามารถใส่ค่าทศนิยมได้
การแสดงผลทั้งหมด แสดงผลบน 16x2 LCD Display
รายละเอียดโครงงาน
ส่วนประกอบในชิ้นงาน 
  • Keypad 4x4
  • Arduino UNO
  • 16x2 LCD Display
  • Breadboard
  • Connecting wires
วิธีการใช้งาน
ให้กดตัวเลขตัวแรกที่อยากจะนำมาใช้ในการคำนวณ จากนั้นกดตัวดำเนินการ (+ - * หรือ /) แล้วกดตัวเลขตัวที่สอง แล้วกดปุ่ม # ในการคำนวณ
ผลลัพธ์ของการคำนวณจะอยู่บนหน้าจอแสดงผล 16x2 LCD Display
